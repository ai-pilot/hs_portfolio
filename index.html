Production RAG with guardrails. Documents are processed through parent-child chunking for context preservation, embedded and indexed in a vector database. At query time, hybrid search (dense + sparse) with Reciprocal Rank Fusion retrieves candidates, which are reranked for precision. The LLM generates grounded responses with source citations, filtered through NeMo Guardrails for safety and relevance.

Agentic orchestration with LangGraph. A central orchestrator routes tasks to specialized agents — each with its own tools and context. Tools are exposed via MCP (Model Context Protocol) for standardized tool integration, while A2A (Agent-to-Agent) protocol enables inter-agent communication across systems. The Retriever handles knowledge queries via RAG, the Validator enforces business rules, the Analyzer runs ML models, and the Reporter formats outputs. A human-in-the-loop gate ensures critical decisions get reviewed, while guardrails maintain safety boundaries throughout.
